最近在 Github Trending 里发现了这个项目，突然发现自己几个月前就搞过这样的小玩具，没想到竟然有另一个人也给做出来了，还有 7.5k star。

其实它的原理也非常简单，就是把数据按 0 和 1（或者映射成 RGB）编码成一帧帧的类二维码图，然后组成视频传到公共视频平台上。从另一个的角度来说，这相当于把视频平台当成了一个超大（甚至无限大）的 OSS，只不过这个 OSS 的存储和读取步骤较为繁琐。
